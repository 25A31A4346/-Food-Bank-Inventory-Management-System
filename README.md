🍱 Food Bank Inventory Management System
The Food Bank Inventory Management System is a C++ based mini project developed to support the United Nations Sustainable Development Goal – Zero Hunger 🌍. This system helps food banks, NGOs, charities, and community kitchens manage donated food items in an organized and efficient way.

The main objective of this project is to reduce food wastage and ensure proper food distribution to people in need. The system stores important details such as Food ID, Food Name, Quantity, and Expiry Date. Users can easily add new food items, search available stock, update quantities, view complete inventory records, and delete expired or unavailable items.

✨ Key Features:

➕ Add Food Items
🔍 Search Food Records
📋 View Complete Inventory
✏️ Update Food Quantity
❌ Delete Food Items
📦 Manage Food Stock Efficiently
This project is developed using:

💻 C++
🧠 Object-Oriented Programming (OOP)
📚 Arrays and Functions
🔄 Loops and Conditional Statements
The system provides a simple menu-driven interface that is easy to understand and user-friendly for beginners. It demonstrates how programming can be used to solve real-world social problems like hunger management and food distribution.
